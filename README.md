<!-- Run Locally -->
### :running: Run Locally


Install dependencies

```bash
  npm install express
```
```bash
  npm install -D tailwindcss
```
```bash
  npm install framer-motion
```

Start the server

```bash
  npm run build
```

```bash
  node server.js
```
